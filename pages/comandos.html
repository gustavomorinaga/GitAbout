Principais Comandos do Git

Nesta página, você verá os principais comandos do Git:

  • git init
    Cria um novo repositório;

  • git status
    Apresenta o status do repositório;

  • git add nome _do_arquivo.txt
    Adiciona um novo arquivo especifico;

  • git add nome _do_diretorio
    Adiciona um novo diretório especifico;

  • git reset
    Reverte o git add;

  • git commit meu_arquivo.txt
    Comitar um arquivo;

  • git commit meuarquivo.txt -m "minha mensagem de commit"
    Comitar informando uma mensagem;

  • git rm meu_arquivo.txt
    Remove arquivo;

  • git rm – r diretório
    Remove o diretório;

  • git log
    Para visualizar o histórico de commits;

  • git push
    Envia as mudanças para o servidor;

  • git pull
    Captura as mudanças que estão salvas no servidor;
